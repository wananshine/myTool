# vue-dropload下拉加载实例

## 项目目的
* 主要用以测试vue-resource，vue-router测试单页应用

## 相关知识点
* .vue组件的开发
* vue-router 简单路由的使用
* 项目中公共样式如何引入
* vue-resource 进行ajax请求的使用
* 使用babel以及webpack 进行编译打包
* 模态框
* 轮播组件

## 安装
* 本例使用vue-cli开发的，所以可以直接clone到本地
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
